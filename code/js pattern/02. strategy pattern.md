##  策略模式：

 **核心**： 定义一系列算法，把它们一个个封装起来，并且让它们可以相互替换。  
 **适用情景**： 可以使用If or switch 的多分支场景。

本质为 将算法的实现和算法的使用分开来。


**策略模式实现**：  

第一是 策略类：  
   在javascript 中可以使用对象中不同的函数属性来实现：
~~~
   var strategy = {

       alg01 : function(){},
       alg02 : function(){},
       ...
   };
~~~

第二是 环境类context，也即调用对象  
     context对象接受客户请求，然后将请求委托给某一个策略函数。


**扩展**

策略模式是用来封装算法的，将算法扩展开来，也可以用来封装一系列的“业务规则”。
